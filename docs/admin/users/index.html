      Пользователи

      Вкладка «Пользователи» отображает всех зарегистрированных пользователей с информацией о токенах, устройствах и сроках доступа.

      Счётчик

      В верхней части вкладки отображается счётчик:

      • Всего — общее количество пользователей
      • Активных — пользователи с действующим токеном
      • Истекших — пользователи с истёкшим сроком

      Таблица пользователей

      Для каждого пользователя показывается:

      ПолеОписание
      TelegramИмя пользователя и ID в Telegram
      ТокенУникальный UUID-токен доступа
      СозданДата создания токена
      ИстекаетДата окончания действия
      УстройстваСписок привязанных устройств (UID, имя, IP, fingerprint, дата привязки, последнее использование)

      Информация об устройствах

      Каждое устройство в карточке пользователя показывает:

      • UID — уникальный 8-символьный идентификатор (lampac_unic_id)
      • Имя — название устройства (Chrome, Lampa TV и т.д.)
      • IP — последний известный IP-адрес
      • Fingerprint — хэш параметров устройства (разрешение экрана, часовой пояс, язык, платформа)
      • Привязано — дата привязки
      • Последний запрос — дата последнего обращения к серверу

      IP и fingerprint обновляются автоматически при каждом запросе от устройства.

      Действия

      Создание токена

      Кнопка «Создать токен» генерирует новый токен доступа. Можно указать срок действия (по умолчанию 30 дней).

      Продление доступа

      Кнопка «Продлить» позволяет увеличить срок действия токена на указанное количество дней.

      Удаление пользователя

      Кнопка «Удалить» полностью удаляет пользователя и все его токены.

      Управление устройствами

      • Отвязать устройство — удаляет привязку конкретного устройства
      • Установить лимит — изменяет максимальное количество устройств для пользователя (0 = без ограничений)

      Управление TorrServer

      Кнопка «🧲 TS» в карточке пользователя переключает доступ к TorrServer:

      • TS: ✓ Вкл (зелёный) — пользователь может использовать TorrServer
      • TS: ✗ Выкл (красный) — доступ к /ts/* заблокирован (403)

      Используется для ограничения ресурсоёмкого торрент-стриминга конкретным пользователям.

      Быстрая блокировка

      В карточке каждого пользователя и устройства доступны кнопки быстрой блокировки:

      • 🚫 TG — заблокировать Telegram ID пользователя
      • 🚫 UID — заблокировать конкретное устройство
      • 🚫 IP — заблокировать по IP устройства
      • 🚫 FP — заблокировать по fingerprint устройства

      Блокировки управляются на вкладке Блокировки.

      Настройка балансеров (Kit)

      Если включена система Kit, для каждого пользователя доступна настройка видимости балансеров — какие источники показывать конкретному пользователю.

      Поиск

      Поле поиска над таблицей позволяет фильтровать пользователей по:

      • Имени пользователя в Telegram
      • Telegram ID
      • UID устройства

      Поиск работает в реальном времени — результаты обновляются по мере ввода.

      Пагинация

      При большом количестве пользователей таблица разбивается на страницы. Доступен выбор количества записей на странице:

      • 10 — компактный вид
      • 50 — стандартный
      • 100 — расширенный
      • Все — без пагинации
