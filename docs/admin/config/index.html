      Конфиг

      Вкладка «Конфиг» предоставляет прямой доступ к редактированию файла конфигурации init.conf через веб-интерфейс.

      Редактор

      JSON-редактор с подсветкой синтаксиса. Позволяет редактировать любые параметры конфигурации вручную.

      Сохранение

      При сохранении:

      1. Проверяется валидность JSON-синтаксиса
      2. Конфигурация записывается в init.conf
      3. Часть параметров применяется сразу (балансеры, плагины)
      4. Некоторые параметры требуют перезапуска (порт, TG-бот)
      Будьте осторожны при ручном редактировании — некорректный JSON может привести к ошибке запуска. Рекомендуется использовать специализированные вкладки (Балансеры, Плагины, Прокси) для типовых настроек.

      Справочник

      Полный список всех полей конфигурации — в разделе Справочник init.conf.
